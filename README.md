# Academic Pages Site

Self-contained Jekyll academic website template for GitHub Pages using GitHub Actions.

## Quick start

1. Replace `yourusername` in `_config.yml`.
2. Replace `files/CV.pdf` with your real CV.
3. Replace `images/profile.png` with your profile image.
4. Push to GitHub.
5. In GitHub: **Settings → Pages → Build and deployment → Source → GitHub Actions**.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```
